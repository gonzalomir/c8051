c8051
=====

Ejemplos de Programacion en C para la familia de microntronladores 8051, usando compiladores MikroC, Keil y SDCC

 - Introducción al Lenguaje C
 - Sintaxis
 - Tipos de Datos (bit, char, int, long, float, ...)
 - Estructuras de Control (if, select case)
 - Estructuras Repetitivas (for, while, do...while)
